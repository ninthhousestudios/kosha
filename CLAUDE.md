# kosha

Document intelligence layer for the manas ecosystem. Decomposes documents into segments, embeds them into a shared vector space, and exposes semantic search and citation tools over MCP.

## Agent skills

### Issue tracker

Issues tracked in yojana (local MCP task graph). See `docs/agents/issue-tracker.md`.

### Triage labels

Default vocabulary (needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout. See `docs/agents/domain.md`.
